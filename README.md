欢迎大家使用【魔法分享】！

# tips
- 支持从粘贴板分享链接：Android 7.1 的用户可以用到App Shortcut，其他版本可以从主界面找到菜单选项；
- 分享对话框点击图片可以进到图片浏览模式，可以用来浏览、下载和分享网页上的图片；
- 点击网页标题可以打开页面；
- 目前还是非常非常早的版本；
